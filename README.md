# 이어버스

실시간 버스 정보 누락을 대비하여 기계학습 모델을 통해 예측정보를 제공하는 버스 정보 시스템 애플리케이션.
Flutter를 통한 iOS와 Android를 지원하는 크로스플랫폼 애플리케이션.
